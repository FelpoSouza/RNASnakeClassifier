# RNASnakeClassifier
Repositório do projeto de Redes Neurais sobre classificação de cobras quanto a sua espécie e família.
As imagens são rotuladas quanto à espécie e família, sendo a primeira dependente da segunda e logo gerando uma hierarquia.
O arquivo principal é o "TesteHML4.py" sendo os outros auxiliares para formar o dataset.
